7 Skills Thực Tế Không Thể Thiếu

Mỗi skill giải quyết một bài toán thực tế — tiết kiệm hàng giờ làm việc mỗi tuần

🌅

Skill #1 – Morning Briefing

Mỗi sáng gõ "morning briefing" → Claude quét lịch, email, tin tức AI → tạo dashboard web đẹp với mọi thứ cần biết để bắt đầu ngày. Tích hợp Slack → nhận brief ngay trên điện thoại trước khi mở máy. Tiết kiệm 30 phút mỗi sáng.

GmailCalendarSlackHTML Dashboard
🔬

Skill #2 – Research Assistant

Gõ "research assistant [chủ đề]" → Claude tìm kiếm web, tổng hợp, ghi nguồn đầy đủ → tạo báo cáo markdown hoàn chỉnh có thể chia sẻ. Thay thế 30-60 phút research thủ công. Mỗi claim đều có link nguồn kiểm chứng.

Web SearchTổng hợp nguồnFile Markdown
📝

Skill #3 – Meeting Notes → Action Items

Paste transcript họp vào → Claude tóm tắt → liệt kê action items từng người + deadline + câu hỏi còn mở → lưu file. Cuộc họp 1 tiếng thành 10 hành động rõ ràng. Không bao giờ rời phòng họp mà không biết làm gì tiếp.

TranscriptAction ItemsDeadline
🖥️

Skill #4 – Slide Deck Generator

Gõ một câu mô tả chủ đề → nhận bộ slides HTML đầy đủ với animation đẹp, visual mỗi slide, thiết kế chuyên nghiệp. Tiết kiệm 3 tiếng làm presentation. Tất cả slides trong khóa học gốc được tạo bằng skill này.

HTML SlidesAnimationDesign chuẩn
🎨

Skill #5 – Visual Explainer

Mô tả concept → Claude tạo trang web tương tác với diagram, animation, typography đẹp. Hoàn hảo để giải thích quy trình phức tạp, system architecture, business flow. Dùng để visualize toàn bộ hệ thống Cowork của bạn.

Interactive WebDiagramAnimation
📐

Skill #6 – Diagram Generator

Mô tả thứ cần vẽ → Claude tạo file JSON định dạng Excalidraw → import vào Excalidraw ngay lập tức. Không cần vẽ tay hay dùng PowerPoint shapes. Hoàn hảo cho flowchart, system architecture, mind map.

Excalidraw JSONFlowchartSơ đồ
⚙️

Skill #7 – Skill Creator

Skill tạo ra Skill khác! Mô tả quy trình muốn tự động hóa → Claude hỏi thêm câu hỏi → đóng gói thành skill hoàn chỉnh. Dù bạn là giáo viên, môi giới BĐS, designer hay freelancer — đều tạo được skill riêng cho mình.
